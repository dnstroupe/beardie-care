# 🦎 Bearded Dragon Care Command Center

A premium, mobile-friendly care tracker and feeding calendar for your bearded dragon.
Features a sidebar navigation, bearded dragon photos throughout, weekly calendar with feeder images, and a full desert canyon theme.

---

## 📁 Files in This Project

```
beardie/
├── index.html   — Main page (all sections)
├── style.css    — Desert-canyon theme, sidebar layout, responsive
├── script.js    — Checkboxes, localStorage, weekly calendar, sidebar
└── README.md    — This file
```

---

## ✅ Features

- **Sidebar navigation** with bearded dragon portrait photo
- **"Good morning/afternoon/evening!"** hero with large dragon photo + live progress card
- **Daily Morning Checklist** — 10 tasks with live progress bar
- **Weekly Feeding Calendar** — Full table with feeder images, quantity badges, checkable rows per day
- **Today auto-highlighted** in terracotta with a ★ badge
- **Reset Week** and **Today** scroll buttons
- **Monthly Maintenance** — Collapsible, 8 checkable tasks
- **Master Shopping List** — Collapsible, 3 columns (greens, veggies, feeders)
- **Supplements Owned** — Color-coded reference with schedule
- **Emergency Guide** — Collapsible with bearded dragon photo
- **Family Notes** — Collapsible with desert landscape photo
- **Cactus silhouette footer**
- **All checkboxes saved in localStorage** — survive page refresh
- **Mobile responsive** — hamburger menu, single-column stacking

## 🚀 Deploy to GitHub Pages (Step-by-Step)

### Step 1 — Create a GitHub Account
1. Go to **https://github.com**
2. Click **Sign up** and create a free account
3. Confirm your email

### Step 2 — Create a New Repository
1. Click **+** (top right) → **New repository**
2. Name it: `beardie-care` *(no spaces)*
3. Set to **Public**
4. ✅ Check **"Add a README file"**
5. Click **Create repository**

### Step 3 — Upload Your Files
1. Click **Add file** → **Upload files**
2. Drag and drop all 4 files: `index.html`, `style.css`, `script.js`, `README.md`
3. Click **Commit changes**

### Step 4 — Turn On GitHub Pages
1. Click **Settings** tab
2. Left sidebar → **Pages**
3. Under Branch → select **main** → folder **/ (root)**
4. Click **Save**
5. Wait 1–2 minutes, then your URL appears:
   ```
   https://YOUR-USERNAME.github.io/beardie-care/
   ```

### Step 5 — Share With Your Wife
1. Copy the URL and text it to her
2. **iPhone:** Safari → Share → Add to Home Screen
3. **Android:** Chrome → ⋮ Menu → Add to Home Screen

---

## 🔒 Privacy

All checkbox data is saved in your browser's localStorage — it never leaves your device.
You and your wife each have independent progress on your own phones.
